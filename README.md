# Паттерны проектирования GoF на Java

Набор минимальных и практичных примеров классических паттернов проектирования (Gang of Four) на Java c пояснениями.

---

## О чём репозиторий

- **Цель:** показать рабочие, короткие и понятные реализации паттернов без «магии».
- **Формат:** по одному паттерну в пакете, лаконичный код + комментарии.
- **Фреймворки:** чистая Java (без зависимостей на DI-контейнеры), чтобы фокус был на идее.
